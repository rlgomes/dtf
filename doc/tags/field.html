Field

The field tag is just like the attribute tag but is used in the context of queries to identify the fields that are suppose to selected by the select clause in the query.



Author
Rodney Gomes


Optional Attributes

type

Identifies the type of attribute being recorded.

Supported DTF types:
    Type Description
    int Integer type
    long Long type
    string String type



Required Attributes

name

name of the property to define.
