Call

Execute functions that were defined in this test case or imported using the

Import

tag. Any properties created/altered within a function will not be visible after this function has terminated its execution.



Author
Rodney Gomes


Children Tags

(property){0,*}


Optional Attributes

result

the property to store the return value from this function.



Required Attributes

function

The unique name of the function to call from this point in the XML testcase.


Usage Examples

Example #1
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<sequence>
     <function name="func3">
         <param name="nomore" type="optional"/>  
         <local>
            <echo>In func3</echo>
        </local>
     </function>
     
     <call function="func3"/>
     
     <call function="func3">
         <property name="nomore" value="true"/>
     </call>
 </sequence>

Example #2
