Getfile

this file can be used to retrieve a file from a component and place it in a storage identified on the runner side.



Author
Rodney Gomes


Optional Attributes

append

Specifies if you would like the file to append to any existing data in the uri specified.

compress

Specifies if you would want the file to be compressed at the destination. This also speeds up the copy because the compression is done on the agent side.



Required Attributes

uri
remotefile

The fullpath to the remote file relative to where the DTFA was started.


Usage Examples

Example #1
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<getfile remotefile="build.xml" uri="storage://OUTPUT/build.copy1.xml"/>

Example #2
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<getfile remotefile="/etc/hosts" uri="storage://OUTPUT/hosts1"/>
