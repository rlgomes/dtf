Input

This tag is used to just point to inputs for other tags such as the

diff

tag which uses this tag to reference the data that the test writer wants to calculate the differences on. See

diff

for more examples.



Author
Rodney Gomes


Optional Attributes

uri

The URI of the data that will be used as an input.


Usage Examples

Example #1
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<diff>
     <input uri="storage://OUTPUT/file1"/>
     <input uri="storage://OUTPUT/file2"/>
     <input uri="storage://OUTPUT/file3"/>
 </diff>
