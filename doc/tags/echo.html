Echo

The echo tag is used to print out test case information during the execution of the test case. It is preferred that you use the tag

log

instead of echo since you can easily set the logging level and that way have control over what is being logged in your test run just by editing the dtf.properties file and changing the log4j properties.



Author
Rodney Gomes


Optional Attributes

message

The message attribute contains the message to be logged to the test execution log.


Usage Examples

Example #1
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<echo message="Hello World"/>

Example #2
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<echo>Hello World</echo>
