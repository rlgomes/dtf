Log

Logging ability to be able to place messages in the test run that are similar to the ones that are logged within the test code itself.



Author
Rodney Gomes


Optional Attributes

tag

The tag will be used when logging as the first tag in the logline that is being logged. For example: 

                [java] ERROR 15/04/2008 09:05:33 YOURTAGHERE          - your message
                [java] INFO  15/04/2008 09:05:33 YOURTAGHERE          - your message

level

Specifies the level of logging to be printed, this can be one of the 3 following:
Log Level Description
info Info log level is used in normal conditions.
warn Warning log level is used in conditions that may require attention.
error Error log level is used in error conditions, and usually require attention.


Usage Examples

Example #1
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<log level="warn">Attention!</log>

Example #2
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<log level="debug" tag="MYDEBUG">Attention!</log>

Example #3
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<log level="error">Something has gone horribly wrong.</log>
