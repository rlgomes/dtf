If

The if tag will execute the then tag if the condition is true and if the condition evaluates to false then the else sub tag will evaluate to true.



Author
Rodney Gomes


Children Tags

(false | true | not | and | eq | gt | lt | neq | or | within | match | isset | rendezvous_check | rendezvous_exists | share_exists) . then . else

Usage Examples

Example #1
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<if>
     <eq op1="${test1}" op2="${test2}"/>
     <then>
         <local>
             <echo>${test1} equals ${test2}</echo>
         </local>
     </then> 
     <else>
         <local>
             <fail message="CRAP!"/>
         </local>
     </else>
 </if>

Example #2
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<if>
     <neq op1="${test1}" op2="${test2}"/>
     <then>
         <local>
             <echo>${test1} is not equals ${test2}</echo>
         </local>
     </then> 
 </if>
