Unset

This tag simply unsets the property defined by the attribute property from the DTF state. This allows you to make a property dissappear from execution.



Author
Rodney Gomes


Optional Attributes

property

The name of the property to unset.


Usage Examples

Example #1
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<unset property="blah"/>
