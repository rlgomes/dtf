Rendezvous_destroy

destroy the specified rendezvous cleaning up any resources.



Author
Rodney Gomes


Required Attributes

id

the unique identifier used to reference this particular rendezvous point within a DTF testcase.


Usage Examples

Example #1
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<local>
      <rendezvous_destroy id="myrendezvous1"/>
 </local>
