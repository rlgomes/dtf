Testscript

The testscript tag allows you to execute an external XML script by identifying where it is with the uri attribute. The properties,references are inherited from the parent script, but all components and storages are not inherited from the parent script.

This tag generates test result events that can be recorded with the results tag and later processed to keep track of test results over of time or builds.



Author
Rodney Gomes


Children Tags

(testproperty){0,*}


Required Attributes

uri

The uri identifies the location of the script to be executed.


Usage Examples

Example #1
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<testscript uri="storage://INPUT/storage.xml"/>
