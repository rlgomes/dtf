Header
this tag is used within the context of an http request and will define a simple header with a name and value.
Required Attributes
name
the name of the header being specified.
value
the value of the header being specified.
Usage Examples
Example #1
<http_get onFailure="fail" perfrun="true" uri="${dtf.http.uri}">
   <header name="header" value="XXXXX" />
</http_get>
