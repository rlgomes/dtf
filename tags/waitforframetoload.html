Waitforframetoload

Waits for a new frame to load. Selenium constantly keeps track of new pages and frames loading, and sets a "newPageLoaded" flag when it first notices a page load. See waitForPageToLoad for more information.



Author
Rodney Gomes


Required Attributes

frameAddress

frameAddress from the server side

timeout

the amount of time in milliseconds to wait for a page to load after which an error will be thrown and the current execution would terminate.
