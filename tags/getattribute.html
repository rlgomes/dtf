Getattribute

Gets the value of an element attribute.

Required Attributes
attributeLocator
an element locator followed by an @ sign and then the name of the attribute, e.g. "foo@bar".
property
the name of the property to save the value to.
Usage Examples
Example #1
<selenium baseurl="http://www.yahoo.com" browser="*firefox">
   <open url="/" />
   <getAttribute attributeLocator="//button[@value='Web Search']@id" property="search_id" />
</selenium>
