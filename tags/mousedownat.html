Mousedownat

This command simulates a mouse down event at the location specified by the locator attribute using the Element Locator syntax. The location of the click can also be influenced by using the coordString to identify the specific position within the element identified by the locator.



Author
Rodney Gomes


Required Attributes

locator

Element Locator to identify the html element.

coordstring

Usage Examples

Example #1
<selenium baseurl="http://www.google.com" browser="*firefox">
      <open url="/"/>
      <mouseDownAt coordString="0,0" locator="//input[@value='q']"/>
  </selenium>
