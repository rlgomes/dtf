Log

Logging ability to be able to place messages in the test run that are similar to the ones that are logged within the test code itself.



Author
Rodney Gomes

Usage Examples

Example #1
<log level="warn">Attention!</log>

Example #2
<log level="debug" tag="MYDEBUG">Attention!</log>

Example #3
<log level="error">Something has gone horribly wrong.</log>
