Iselementpresent

Verifies that the specified element is somewhere on the page.



Author
Rodney Gomes


Required Attributes

locator

Element Locator to identify the html element.


Usage Examples

Example #1
<selenium baseurl="http://www.google.com">
      <open url="/"/>
      <waitForPageToLoad timeout="30000"/>
      <assert>
        <isElementPresent locator="q"/>
    </assert>
  </selenium>
