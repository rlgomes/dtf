Refresh

This command allows you to refresh the the currently focused window.



Author
Rodney Gomes

Usage Examples

Example #1
<selenium baseurl="http://www.yahoo.com" browser="*${browser}">
      <open url="/finance"/>
      <waitForPageToLoad timeout="30000"/>
      <refresh/>
      <waitForPageToLoad timeout="30000"/>
  </selenium>
