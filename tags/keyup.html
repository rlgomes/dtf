Keyup

This command simulates releasing a specified key sequence.



Author
Rodney Gomes


Required Attributes

locator

Element Locator to identify the html element.

keySequence

either be a string("\" followed by the numeric keycode of the key to be pressed, normally the ASCII value of that key), or a single character. For example: "w", "\119".


Usage Examples

Example #1
<selenium baseurl="http://www.yahoo.com" browser="*firefox">
      <open url="/"/>
      <keyUp keySequence="w" locator="//input[@name='p']"/>
  </selenium>
