Return
Return a value to the call tag from within a function. For the time being this return statement can only be done at the end of the function.
Usage Examples
Example #1
<sequence>
   <function name="func3">
      <param name="flag" type="required" />
      <return>${succeeded}</return>
   </function>
   <call function="func3" result="result">
      <property name="flag" value="true" />
   </call>
   <log>func3 returned ${result}</log>
</sequence>
