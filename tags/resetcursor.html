Resetcursor

The reset cursor places the cursor back at the start where it was immediately after the

query

tag was called. This allows you to reuse the same cursor and not have to re-issue the same query.



Author
Rodney Gomes


Required Attributes

cursor

Identifies the cursor that should be reset so that it seems as if the cursor was reopened by a

Query

tag.


Usage Examples

Example #1
<?xml version="1.0" encoding="UTF-8" standalone="no"?>
<resetcursor cursor="cursor2"/>
