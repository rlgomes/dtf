Sort
This tag will sort an already existing recording of events by the field specified and will rewrite the file using the same previously recorded format. In the process of sorting you can also filter out the fields that no longer interest you and save time in the whole sorting process.
Children Tags
field . select
Required Attributes
input
the uri of the input of events to be read from.
output
the uri of the output of events to be written to.
event
the exact event name if you wish to order (and filter) on a specific set of events.
