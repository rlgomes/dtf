Click

This command simulates a single click at the location specified by the locator attribute using the Element Locator syntax.



Author
Rodney Gomes


Required Attributes

locator

Element Locator to identify the html element.


Usage Examples

Example #1
<selenium baseurl="http://someplace.com" browser="*firefox">
      <open url="/"/>
      <click locator="//input[@name='test']"/>
  </selenium>
