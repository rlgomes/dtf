Abs

Calculates the absolute value of a number, converting all numbers to a positive value and placing that in the property identified by the attribute result.



Author
Rodney Gomes


Required Attributes

op1

first operand of the arithmetic operation.

result

property name of where to store the result of the arithmetic operation.


Usage Examples

Example #1
<abs op1="${prop1}" result="result"/>

Example #2
<abs op1="${prop1}" result="prop1"/>
